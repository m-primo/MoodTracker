# Mood Tracker
A personal python project to create a mood tracker app

# How to
1. First you need to create the database and its tables, so you'll need to migrate:

   `python migrate.py`
2. Then run the app as you preferred, whether the GUI or the CLI:

   `python cli.py`

   `python gui.py`
